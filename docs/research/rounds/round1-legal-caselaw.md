## 1. ロクラクII事件（最高裁 平成23年1月20日 第一小法廷判決）

### 1.1 基本情報

| 項目 | 内容 |

| 判決日 | 平成23年（2011年）1月20日 |
| 法廷 | 最高裁第一小法廷（裁判長 金築誠志） |

| 主文 | 原判決破棄・知財高裁へ差戻し（全員一致） |
| 参照条文 | 著作権法21条（複製権）、98条（放送事業者の複製権） |

### 1.2 事案の概要

被上告人はインターネット通信機能を有するHDDレコーダー「ロクラクⅡ」を製造・販売・貸与。親機は地上波チューナー内蔵で受信番組を録画しインターネット送信、子機は遠隔地（海外等）から録画指示・受信再生。親子機は1対1対応。初期登録料3,150円、レンタル料月額6,825円〜8,925円のサービスとして提供。放送事業者らが複製権侵害を主張。

### 1.3 「複製の主体」の判断枠組み（規範的主体論）— 判決文原文の規範

> 「複製の主体の判断に当たっては、**複製の対象、方法、複製への関与の内容、程度等の諸要素を考慮して、誰が当該著作物の複製をしているといえるかを判断するのが相当**」

本件へのあてはめ（**「枢要な行為」論**）:

> 「放送番組等の複製物を取得することを可能にするサービスにおいて、サービスを提供する者が、**その管理、支配下において**、テレビアンテナで受信した放送を複製の機能を有する機器に入力していて、当該複製機器に録画の指示がされると放送番組等の複製が自動的に行われる場合には、**その録画の指示を当該サービスの利用者がするものであっても、サービス提供者はその複製の主体である**と解するのが相当である」

理由づけ: サービス提供者は複製を容易にする環境整備にとどまらず、「**複製の実現における枢要な行為**」をしており、これがなければ利用者の録画指示だけでは「複製をすることはおよそ不可能」だから。

### 1.4 金築誠志裁判官の補足意見（原文確認）

1. カラオケ法理（クラブキャッツアイ事件・昭和63年3月15日）は「管理、支配と利益の帰属という二要素を中心に総合判断」してきたが、法的根拠・要件の曖昧さに批判あり
2. 「『カラオケ法理』は、**法概念の規範的解釈として、一般的な法解釈の手法の一つにすぎない**」。複製等の主体判断は「**社会的、経済的側面をも含め総合的に観察すべき**」
3. 「管理、支配と利益の帰属という**二要素を固定的なものと考えるべきではない**」
4. 本件では利益帰属の評価が結論を左右するわけではない。本件システムを「単なる私的使用の集積とみることは、実態に沿わない」

※補足意見の原文に「手足」という文言は登場しない（原文確認済み）。手足論との対比は学説・解説上の整理。

### 1.5 審級の経過

- 第一審（東京地裁）: 侵害肯定（カラオケ法理）
- 原審（知財高裁 平成20年(ネ)10055号・10069号）: 「複製を容易にするための環境等を提供しているにすぎず」として侵害否定（利用者の私的複製と構成）
- 最高裁: 破棄・差戻し。「録画の指示が利用者によってなされるという点にのみに重点を置くことは、相当ではない」（金築補足意見）

### 2.2 事案の概要

ソニー製「ロケーションフリー」のベースステーション（チューナー内蔵・受信放送をデジタル化し自動送信）を利用者が自己所有のまま被上告人に送付、被上告人が事業所内に設置しアンテナ・ネットに接続。利用者は手元端末と1対1対応で海外等から日本の放送を視聴。入会金31,500円・月額5,040円。

### 2.3 「1対1」送信でも公衆送信となる論理（判決文原文）

**(a) 装置該当性**: 送信可能化規制の趣旨は「準備段階の行為の規制」。

> 「公衆の用に供されている電気通信回線に接続することにより、当該装置に入力される情報を受信者からの求めに応じ自動的に送信する機能を有する装置は、**これがあらかじめ設定された単一の機器宛てに送信する機能しか有しない場合であっても、当該装置を用いて行われる送信が自動公衆送信であるといえるときは、自動公衆送信装置に当たる**」

**(b) 送信主体**: 装置に「継続的に情報が入力されている場合には、当該装置に情報を入力する者」が送信主体。ベースステーションを利用者が所有していても、アンテナ接続・設置・管理を行う**被上告人が送信主体**。

**(c) 公衆該当性**: 公衆送信は「**送信の主体からみて**」判断。「何人も」契約すれば利用できる以上、送信主体たる被上告人から見て利用者は不特定の者＝公衆。

→ 個々の機器と端末の1対1対応ではなく、**送信主体を規範的に確定したうえで、その主体から見て受信者全体が公衆か**で判断する枠組み。

### 2.4 その後

差戻審で侵害認定、サービス終了。ソニーもロケーションフリー生産終了。

## 3. カラオケ法理（クラブキャッツアイ事件）と規範的主体論・手足論

- **クラブキャッツアイ事件**（最高裁 昭和63年3月15日・民集42巻3号199頁）: スナック経営者がカラオケ装置設置・従業員が歌唱を勧め集客に利用 → ①管理（支配）性 ②営業上の利益性 の二要素で、客の歌唱を経営者による演奏と同視（利用主体拡張法理）
- **手足論**: 独立の意思・利益なき物理的行為者（従業員等）の行為を背後者に帰属させる伝統的手法。カラオケ法理は独立の意思を持つ第三者（客）まで拡張した点で異なる
- **ロクラクII以後**: 二要素の固定的定式は採らず「諸要素を考慮」する総合考慮型へ相対化。音楽教室事件最高裁判決（令和4年10月24日）では生徒の演奏について教室の主体性が否定され、射程はさらに限定的に

## 採譜SaaSへの含意（プロジェクト向け整理）

- サーバー側で音源を受信・複製・変換するSaaSは、ロクラクIIの「枢要な行為」論により**事業者が複製主体と認定されるリスクが高い**（「ユーザーの私的複製ツール」の建付けは危うい）
- まねきTVの論理により、ユーザーごとの1対1処理であっても「事業者から見て公衆」に対するサービスは公衆送信等の規制対象になり得る
- ただし音楽教室事件（令和4年）で規範的主体論の射程は限定される方向もあり、**設計次第（完全ローカル処理: Neuratron AudioScore型）でリスク回避の余地**がある

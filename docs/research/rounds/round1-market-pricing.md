### A-3. ココナラとの比較(まとめ)

- **最安値はココナラ(1,500円〜)とSKIMA(1,000円〜)**。知恵袋でも「どこの採譜サイトよりも安い。1,500円から」とココナラが推奨されている。
- **専門業者は1曲1パート5,000円〜1万円台が中心**。知恵袋でも「TAB譜3〜4分の曲でどの業者も1万円前後」「技術料として適正」という認識が定着。
- 業界標準的な相場観(まとめ記事): 中級ピアノ譜(4分)1〜1.5万円、メロディ譜3,000〜5,000円、ピアノ2段+メロディで1.3〜2万円。→ ココナラの価格帯は業者相場の1/2〜1/3。
- **Piascore/ぷりんと楽譜は「受託」ではなく「完成譜のストック販売」**で、単価120〜600円×量のモデル。受託(1件数千〜数万円)とは市場構造が異なる。

## 項目B: 需要側の実態(誰が・なぜ頼むのか)

### B-1. 依頼者の属性と用途(事実ベース)

| 属性 | 用途・エビデンス |
|---|---|
| **ピアノ愛好家(市販譜がない曲)** | 知恵袋「ピアノで弾いてみたい曲がマイナーすぎて楽譜がない」「絶対音感もないし耳コピもできなくて困っている」— 最頻出パターン |
| **バンド(コピバン)** | 市販バンドスコアが存在しない曲・市販譜が間違っているケースで専門業者(弦譜堂・ドリームスコア等)へ。1パートフルコーラス2,500円〜の依頼例 |
| **発表会・コンクール(ピアノ/エレクトーン)** | 「発表会・コンクール用アレンジ」需要が明記。エレクトーン対応者は希少で需要が高い |
| **弾き語り** | SKIMAに「ギター弾き語りをしたい人向けコード採譜(2曲)」の出品。コード譜だけの軽量需要が存在 |
| **合唱・音楽教員** | パート練習用の音取り音源・パート譜作成需要。自作(スコアメーカー)かプロ外注かの比較記事あり |
| **吹奏楽** | 採譜+大編成への編曲(WINDSで15〜30万円)。楽譜代行業者は吹奏楽対応を明記 |
| **作曲者・シンガーソングライター** | 「自分の鼻歌・自作曲を形ある楽譜にしたい」(Kanon)。作詞家が「譜割り」を依頼される例(知恵袋) |
| **演奏動画視聴者** | 「YouTubeで見た演奏を自分も弾きたい」が依頼理由の定番。YouTube動画URLを音源として送る運用が一般化 |
| **カラオケ・歌唱練習** | ガイドメロディ(ハモリ対応)作成、カラオケ全パート耳コピの商業実績を持つ出品者あり |
| **特殊楽器・ニッチ** | 三味線の音源採譜(秀五郎)、中世音楽・民謡の記録、電子オルガン演奏のドラム採譜など |

### B-2. 依頼する動機(生の声)

- **技術的にできない**: 「絶対音感もないし、耳コピ？も出来なくて困っている」(知恵袋)。和音の構成音まで完全に聴き取るのはほぼ不可能という認識。
- **楽譜が市販されていない**: マイナー曲・ゲームBGM・YouTube演奏など「そもそも譜面が存在しない」が最大の動機。
- **時間がかかる**: 「きちんと耳コピしようとするとかなりの集中力と気力、体力を使う」— 自力可能でも外注する層。
- **難易度・編成の調整**: 「バイエル程度に」「ソナチネ程度に」等、演奏レベルに合わせたアレンジ込みの依頼が多い(出品者の献立にも明記)。
- **価格感度**: 「普通このくらいのサービスは最低1万円はするはずが2,000円で満足できた」というレビュー — 低価格帯への強い反応があり、ココナラ系が業者から需要を奪う構図。

### B-3. 事実と推測の区別・制限事項

- 上表の価格はすべてWeb上の公開ページ・検索結果から取得した事実。
- タイムチケットの個別価格はHTTP 403で取得不可(不明)。クラウドワークス・メルカリ・noteの採譜特化相場は定量データが見つからず不明。
- 「ココナラ系が業者需要を奪う構図」「合唱指導者の外注が増加」は状況証拠からの**推測**。
- 需要の絶対量(月間依頼件数など)の公開統計は存在せず、ココナラのカテゴリ実績「5.1万件」が唯一のマクロ指標。

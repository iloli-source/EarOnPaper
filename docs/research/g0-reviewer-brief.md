## レビュアー向け説明

### やっていただくこと

1. 同梱の元音源（`input/` の3曲）を聴く
2. 各曲についてAIツール2種の出力（`output-musescore/` の楽譜ファイル、`output-muscriptor/` のMIDI）を MuseScore Studio（無料: musescore.org）で開いて再生・目視する
3. 曲×ツールごと（計6件）に下の評価シートを記入する

### 評価の観点（この4つだけ見てください）

- **音の高さ**: 元の曲と同じ音が拾えているか
- **リズム**: 音の出だし・長さ・拍子が合っているか
- **譜面の読みやすさ**: 楽譜として人に渡せる見た目か（小節や声部がぐちゃぐちゃでないか）
- **実用度**: この譜面を出発点に手直しして使うのと、ゼロから耳コピするのと、どちらが楽そうか

### 大事なお願い

- 完璧な分析は不要です。**演奏者としての率直な体感**が一番の価値です
- 1件あたり10〜15分を目安に。迷ったら直感で
- MIDIファイル（MuScriptor出力）は譜面の見た目が MuseScore の自動変換に左右されるため、「音の内容」と「譜面の見た目」を分けて感じたままに書いてください

## 評価シート（曲×ツールごとに1枚、計6枚）

```
曲ID: G0-0_ ／ ツール: [ ] MuseScore audio2score  [ ] MuScriptor

1. 音の高さの正しさ        1（全然違う）- 2 - 3 - 4 - 5（ほぼ合っている）
2. リズムの正しさ          1 - 2 - 3 - 4 - 5
3. 譜面としての読みやすさ   1（読めたものではない）- 2 - 3 - 4 - 5（人に渡せる）
4. 実用度: この出力を手直しして使うのは、ゼロから耳コピするのと比べて…
   [ ] かなり楽（半分以下の手間で済みそう）
   [ ] 多少は楽だが、結構直す必要がある
   [ ] ゼロからやったほうが早い（使いものにならない）
5. 崩れ方はどちらが近いですか（複数可）
   [ ] 音そのものが間違っている・拾えていない（高さや出だしがおかしい）
   [ ] 音はだいたい合っているが、譜面が汚い・弾ける形になっていない
   [ ] 同じような間違いが繰り返されている（規則的）
   [ ] 間違い方がバラバラ（不規則）
6. 一言コメント（自由記述）:
```

## 返送とお願い事項

- 記入済みシートはテキスト・写真・口頭メモのどれでも構いません（こちらで清書します）
- 音源・出力ファイルは検証目的の私的な共有です。**第三者への転送やSNS等への公開はお控えください**。終了後は削除していただけると安心です

## FAQ

- **Q. 楽譜ソフトを持っていない** — MuseScore Studio（無料）を musescore.org からダウンロードしてください。開いて再生ボタンを押すだけで音が出ます
- **Q. MIDIを開いたら譜面がすごく汚い** — それ自体が評価対象です。シート5で「譜面が汚い」側にチェックし、音の内容が合っているかは再生で判断してください
- **Q. 曲を全部聴き通す必要がある？** — いいえ。各曲1〜2分ぶん、サビや特徴的な部分の比較で十分です
- **Q. 2つのツールの区別がつかない** — フォルダ名で区別してください。どちらが優れているかの順位づけは不要で、シートが6枚あれば足ります
- **Q. 専門用語で書いたほうがいい？** — 不要です。普段のことばで大丈夫です

# 採譜プロジェクト (AI Music Transcription)

AI採譜（音楽音源 → 楽譜/MIDI変換）のリサーチ・検証・ツール開発・事業化検討を行うプロジェクト。

## フェーズ構成

1. **リサーチ** — 既存ツール・ユーザー体験（成功例/失敗例）・必要機能の網羅調査
2. **実採譜ワークフロー検証** — 実音源での精度・手間の実測（ピアノソロ → ボーカル → フルバンド）
3. **ツール開発** — 検証知見をもとにしたパイプライン/ツール化
4. **事業化検討** — 採譜代行 / SaaS の企画（実測データを根拠に判断）
